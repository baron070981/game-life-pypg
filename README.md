# game-life-pypg
игра "Жизнь" на Python и pygame

----

необходимо, чтоб были установлены:
```
pygame
pygame_widgets
rich
```

---

скачивание:
```
git clone https://github.com/baron070981/game-life-pypg.git
cd game-life-pypg
pip install -r requirements.txt
```

---

запуск:
```
cd project
python main.py
```
